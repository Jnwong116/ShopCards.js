ShopCards.js
Building a shop has never been easier
ShopCards.js is a JavaScript library that makes building and maintaining a shop easy for anyone to do. You can get a clean looking shop up and running in no time.
API Documentation

Item Object

ShopCards needs to receive an array of items for it to work. These items objects describe all aspects of the item you are trying to sell. It is important that you create your item objects in this format for the library to work. Each item object must have a value for each of these attributes listed below:

Attribute Type Description
id String This value must be unique.
name String This should be the name of the item that you wish to display.
price Number This should be the original price of the item.
discount String This value is the discount (if any) applied on the item. It should be a percentage or if there is no discount, it should be the empty string "".
img String This should be a string, with a path to the image you wish to use for the item.
link String This should be the href location of the page you would like the user to be redirected to when clicking on the View More button or when clicking on the item card.
description String This should be a descirption of the item you are selling.
numReviews Number This should be the total number of reviews the item has received.
rating Number This should be a number with the star rating the item has received. It can have a decimal of .5 to show half a star.
An example of an item object ShopCards expects is shown below. 

                    
                        const gpu = {
                            id: 1,
                            name: "RTX 3070",
                            price: 500.0,
                            discount: "",
                            img: "images/items/3070.png",
                            link: "/",
                            description: "This is a RTX 3070 gpu",
                            numReviews: 1002,
                            rating: 3
                        }

If your item has a multiple sizes: then you can add a size attribute to the item to display the different sizes when the user is in quickView. 

                    
                        const soccerJersey = {
                            id: 0,
                            name: "Soccer Jersey",
                            price: 45.0,
                            discount: "",
                            img: "images/items/bayern.jpg",
                            sizes: ["XS", "S", "M", "LG"],
                            link: "/",
                            description: "This is a soccer jersey from FC Bayern",
                            numReviews: 250,
                            rating: 4.5
                        }

Shop Cards

A shopCards object will hold all the data of your shop, including the options you wish to include and any styling choices you wish to make. It also contains all the functions to create the various elements of the shop you may need.

Shop Data Object

This is the object you will need to pass into the shopCards constructor. It contains all the options and styling choices you wish to make.

Attribute Type Description
discountColor String This is the background color of the bubble that will appear on item cards if there is a discount.
quickView Boolean This should be set to true if you want to have quickView enabled. Otherwise set it to false.
items Array This should be an array of the item objects as defined above.
checkoutLink String This should be the location href of your checkout page you, so when users click on the checkout button in the cart overview they will be redirected to the proper place.
ratingColor String This is the color you want the stars to be in the quickView.
reviewColor String This is the color you want the number of reviews to be in the quickView.
addToCartColor String This is the background color of the Add To Cart button.
cartNumberColor String This is the background color of the bubble that appears next to the cart representing how many items are currently in the cart.

An example (and default) version of the shopData object is shown below. 

                    
                        const shopData = {
                            discountColor: "red",
                            quickView: true,
                            items: ItemList,
                            checkoutLink: '/',
                            ratingColor: "green",
                            reviewColor: "green",
                            addToCartColor: "#124cdf",
                            cartNumberColor: "red"
                        }

Constructor

Create a shopCard object by calling the constructor

                    
                        const shop = new shopCards(shopData);

Functions

After you create the shopCards object, you can call upon multiple functions to display and customize your shop as you see fit.

displayItems

  • description - This will create the grid of item cards for you to display on your page.
  • return - It returns a flexbox element with all of the item cards displayed in a grid. You can then go ahead and add any custom styling you would like and then you will need to add it to your page.
  • examples: 

                    
                        const itemGrid = document.getElementById('shopGrid');
                        const shopCardsItemGrid = shop.displayItems();
                        // Add any styling or make changes to the element now
                        itemGrid.appendChild(shopCardsItemGrid);

displayItem

  • description - This will create a singular item card for you.
  • return - It returns the itemCard element You can then go ahead and add any custom styling you would like and then you will need to add it to your page.
  • arguments - It takes in an item object.
  • examples: 

                    
                        const card = shop.displayItem(ItemList[2]);
                        // Add any styling to the card here
                        document.body.appendChild(card);

displayCart

  • description - This will create the cart object for you.
  • return - It returns the cart element with an id of shopCardsCart.
  • examples: 

                    
                        const cartIcon = document.getElementById('cartIcon');
                        const shopCardsCart = shop.displayCart();
                        // Add any styling to the card here
                        cartIcon.appendChild(shopCardsCart);

updateStore

  • description - This will update the store by if any changes are made to the itemList array passed into the ShopCards object.
  • return - It returns nothing
  • examples: 

                    
                        // Deleting an item from the store
                        ItemList.splice(-1, 1);
                        shop.updateStore();

                        // Adding an item to the store
                        ItemList.push(newItem);
                        shop.updateStore();

displayFilters

  • description - This will display the sort functionality at the top of your item grid. Allowing users to sort by price.
  • return - It returns the filter element which you can then style and add to your itemGrid.
  • examples: 

                    
                        const itemGrid = document.getElementById('shopGrid');
                        itemGrid.appendChild(shop.displayItems());
                        const filters = shop.displayFilters();
                        itemGrid.insertBefore(filters, itemGrid.firstChild);
